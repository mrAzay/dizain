Вводные данные

Ключевым стимулом для обращения Эмиля послужили: упадок проектов после пандемии, так как ресторанный бизнес в это время пострадал больше всего. Что было сделано за время

С заказчиком мы провели ХХ сессии по телефону, за время которых мы успели провести следующие шаги:

Проведена сегментация по четырем критериям, утро / день / вечер / ночь / будни / выходные

Сформированы показатели по каждому отдельному промежутку времени. Определены средние показатели и поставлены цели по росту выручки в ночное, вечернее и дневное в выходные и будние дни.

Описаны товары, которые приносят деньги в определенный промежуток времени. Определена выручка по каждой позиции, определена максимальная выработка. Сформулированы цели по каждому из направлений.

Сформулированы гипотезы для быстрого тестирования. Проведено частичное обучение продуктовому подходу.
